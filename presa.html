Se connecter aux machines

Vous pouvez vous connecter en utilisant ssh. Connexion de l'extérieur vers studssh :

ssh identifiant_ucl@studssh.info.ucl.ac.be
Connexion depuis studssh vers une machine de l'infrastructure :

ssh identifiant_ucl@nom_de_la_machine.info.ucl.ac.be
exemple :

ssh identifiant_ucl@skywaker.info.ucl.ac.be

SSH

Pour se connecter aux machines, graphiquement ou non :
  • Générer une paire de clés SSH (privée + publique)
  • Les placer dans le dossier .ssh de votre home sur votre ordinateur
  • Les placer dans le dossier .ssh de votre home sur les machines des salles (le faire sur une seule machine suffit)

SSH

Un tutoriel pour la génération des clés ainsi que eur utilisation est disponible sur le wiki étudiant (https://wiki.student.info.ucl.ac.be/News/FinDeLauthentificationParMotDePasse)
